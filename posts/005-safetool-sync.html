Introducing SafeTool Sync: Smart Folder Synchronization

Presentamos SafeTool Sync: sincronización inteligente de carpetas

May 31, 2026 • By SafeToolHub 31 de mayo de 2026 • Por SafeToolHub

Today we are announcing SafeTool Sync, our third application and a new addition to the SafeToolHub family. Like everything we build, it runs 100% on your device — no internet connection, no cloud uploads, no telemetry.

Whether you're backing up data, mirroring a working directory, or keeping two drives in sync, SafeTool Sync provides a powerful yet simple solution.

What it can do

SafeTool Sync is a cross-platform desktop application for mirror synchronization between disks and folders. It offers:

  • Multiple sync presets: Mirror Exact, Mirror Safe, Copy Only, Mirror Hash, Two-Way Safe, Two-Way Exact, Two-Way Hash, and Custom modes.
  • Smart comparison: Detects modified, renamed, and conflicting files using size, modification time, and SHA-256 hashing.
  • Conflict resolution: Choose how to handle conflicts with flexible policies — Source Wins, Keep Destination, or Manual Review.
  • Smart match: Automatically identifies files that are likely the same but with different names (e.g., report.pdf → report_v2.pdf).
  • Verification: Full or spot-check SHA-256 verification after file operations.
  • Trash safety: Optionally send deleted files to trash instead of permanently removing them.
  • Snapshots: Save and resume sync states between sessions.

The SafeToolHub guarantee

SafeTool Sync is free and open-source software, released under the GPLv3 license with attribution. You can download it, inspect the source code, modify it, and share it. There are no ads, no subscriptions, and no hidden network calls. It does not even contain networking code.

This is the same guarantee we made with SafeTool Pix and SafeTool PDF, and the same guarantee we will make with every tool we build: your data is yours, full stop.

Try it now

SafeTool Sync v1.2.1 is available for Linux, Windows, and macOS. Download it from the GitHub Releases page.

Requirements: Python 3.12+ and the uv package manager. To run: uv sync then uv run python app.py.

Qué puede hacer

SafeTool Sync es una aplicación de escritorio multiplataforma para la sincronización en espejo entre discos y carpetas. Ofrece:

  • Múltiples perfiles de sincronización: Espejo Exacto, Espejo Seguro, Solo Copia, Espejo Hash, Bidireccional Seguro, Bidireccional Exacto, Bidireccional Hash y Personalizado.
  • Comparación inteligente: Detecta archivos modificados, renombrados y en conflicto usando tamaño, fecha de modificación y hash SHA-256.
  • Resolución de conflictos: Elige cómo manejar conflictos con políticas flexibles — Fuente Gana, Mantener Destino o Revisión Manual.
  • Coincidencia inteligente: Identifica automáticamente archivos que probablemente son el mismo pero con nombres diferentes (ej. informe.pdf → informe_v2.pdf).
  • Verificación: Verificación SHA-256 completa o por muestreo después de las operaciones de archivo.
  • Seguridad con papelera: Opcionalmente envía archivos eliminados a la papelera en lugar de borrarlos permanentemente.
  • Instantáneas: Guarda y reanuda estados de sincronización entre sesiones.
