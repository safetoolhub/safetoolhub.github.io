# SafeToolHub Logo Assets

## Available Formats

- **safetoolhub-icon.svg** - Vector format (scalable to any size, ~2.6K)
- **safetoolhub-icon.png** - 1024×1024px with transparency (~51K) - recommended for web
- **safetoolhub-icon-512.png** - 512×512px with transparency (~21K) - for smaller displays
- **safetoolhub-icon.jpg** - 1024×1024px with white background (~22K) - for solid backgrounds

## Recommended Usage

### GitHub
- Repository header: Use `safetoolhub-icon.png` (1024×1024)
- Organization profile: Use `safetoolhub-icon-512.png` or `safetoolhub-icon.png`
- Badges/small displays: Use `safetoolhub-icon-512.png`

### Web
- Logo in navigation: Use `safetoolhub-icon.svg` for best quality at any size
- Social media: Use `safetoolhub-icon.png` (with transparency) or `safetoolhub-icon.jpg`

### Print
- Use `safetoolhub-icon.jpg` for CMYK conversion
- Or use SVG for infinite scalability

## Design Notes

The logo features:
- Gradient background: Indigo to purple (#4f46e5 → #8b5cf6)
- Luminous overlay with radial glow effect
- Semi-transparent shield outline (privacy symbol)
- White wrench icon (tools symbol)
- Rounded square container (512px radius)
